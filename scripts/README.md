# Скрипты для работы с изображениями игр

## Описание

Этот набор скриптов позволяет скачивать изображения игр с внешних источников и обновлять ссылки в коде для использования локальных файлов.

## Файлы

### 1. `download-all-game-images.js`
Скачивает все изображения игр из casino page и сохраняет их в папку `public/game-images/`.

**Возможности:**
- Скачивает 62 изображения игр
- Создает безопасные имена файлов (убирает спецсимволы)
- Пропускает уже существующие файлы
- Создает JSON файл с маппингом (`games-mapping.json`)
- Показывает прогресс скачивания

**Использование:**
```bash
node scripts/download-all-game-images.js
```

### 2. `update-casino-images.js`
Обновляет ссылки на изображения в файле casino page с внешних URL на локальные пути.

**Возможности:**
- Автоматически заменяет все внешние URL на локальные пути
- Использует маппинг из `games-mapping.json`
- Показывает статистику обновлений
- Проверяет, остались ли необновленные URL

**Использование:**
```bash
node scripts/update-casino-images.js
```

## Структура файлов

После выполнения скриптов создается следующая структура:

```
public/
  game-images/
    aviator.jpg
    chicken-road.jpg
    wheel.jpg
    ...
    games-mapping.json
```

## Файл маппинга (games-mapping.json)

Содержит информацию о каждой игре:
```json
{
  "name": "Aviator",
  "filename": "aviator.jpg",
  "localPath": "/game-images/aviator.jpg",
  "originalUrl": "https://...",
  "provider": "PragmaticPlay",
  "category": "Crash"
}
```

## Преимущества локальных изображений

1. **Быстрая загрузка** - изображения загружаются с вашего сервера
2. **Надежность** - не зависят от внешних сервисов
3. **Кэширование** - браузер может кэшировать изображения
4. **SEO** - лучше для поисковой оптимизации
5. **Контроль** - полный контроль над изображениями

## Примечания

- Все изображения сохраняются в формате JPG
- Имена файлов приводятся к нижнему регистру и заменяются дефисами
- Скрипт добавляет задержку между запросами (200ms) для избежания блокировки
- Один файл (Rise of Atlantis) может не скачаться из-за 404 ошибки - это нормально
